# FUTURE_DS_03
Bank Marketing Dataset Analysis - Future Interns Data Science Task 3 Complete EDA and predictive modeling solution for the UCI Bank Marketing dataset (bank-full.csv), analyzing factors driving term deposit subscriptions.
UCI Bank dataset (45k records): EDA + Random Forest model (AUC 0.92) analyzing term deposit subscriptions.

Key Insights: Duration (>300s), previous success (65%), retired jobs (25%) drive 11.7% baseline conversion.

Tech: Python, pandas, scikit-learn, seaborn in Jupyter Notebook.

Files: bank-full.csv (main), bank.csv (sample), bank-names.txt

Results: Feature importance plots, conversion heatmaps, business recommendations saved in outputs.
